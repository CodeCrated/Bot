sg 1




SSH ini menggunakan system cloud


Solusi penggunaan:

Jika tidak support VC, Games dan lainnya, ubah port UDPGW (port UDP / UDP SSH) menggunakan port berikut: 7300 7200 7400 7900


Note:

Jika di log muncul:

Connection lost

HTTP/1.1 400 Bad Request

Status: 500 (Internal Server Error) Server Error

Note:

Jika di log muncul:

Menerima respon: HTTP/1.1 302 Found

Mengirim HTTP status 200 - HTTP/1.1 200 OK

Menerima respon: HTTP/1.1 101 Switching Protocols


INI TANDANYA CONFIG BERFUNGSI DENGAN BAIK & WORK, NAMUN SERVER SSH ATAU JARINGAN OPERATOR YANG SEDANG TIDAK STABIL.


Jika Hal di atas sudah di lakukan, namun tidak konek.

Coba kamu ganti APN (nama akses poin)

Atau kamu bisa mengganti config yang serupa.

Dan bila masih gak konek, tunggu update config selanjutnya.



Jika untuk mengisi SSH pada Aplikasi HTTP Custom atau Aplikasi Dark Tunnel, Rangkai akun SSH nya seperti berikut:

hostname:443@username:password
